# 🛰️ VRCWG Monitor Tool

> ### A powerful, open-source monitoring tool for VRChat
>
> Real-time tracking, logging, OSC display, and advanced player insights — all in one place.

---

## 🔥 What is this?

The **VRCWG Monitor Tool** is designed to give you **full control and visibility** over your **VRChat** sessions.

It tracks everything happening in your instance and turns it into **clean, usable data** — live.

---

## ✨ Features

### 📡 Live Player Monitoring

* Real-time **join & leave tracking**
* Movable window showing **everyone in your current world**
* Accurate detection using VRChat `output_log` (same method as VRCX)

---

### 🧠 Smart Logging System

* Log players with:

  * Reasons
  * Tags
  * Notes
* Automatically merges local + external logs
* Daily updates for latest logged players

---

### 🔄 Name Change Detection

* Detects when a player **changes their username**
* Keeps track of:

  * Current name
  * Previous names
* Helps identify users even if they try to rename themselves 

---

### 👥 Local Friends System

* Add users as **local friends (client-side only)**
* Track:

  * Last seen name
  * Time spent together
  * Meet count
* Fully stored locally — not tied to VRChat’s friend system 

---

### 📜 Player History

* Stores up to **200 recently seen players**
* Quick access for logging or reviewing past encounters 

---

### 🌍 World & Session Tracking

* Detects:

  * World name + ID
  * Instance info (public, friends, invite, etc.)
  * Region + access type
* Generates **launch links** to rejoin worlds
* Tracks how long you've been playing

---

### 💬 OSC Display (Magic Chatbox Style)

* Custom OSC overlay with:

  * Rotating info
  * Custom statuses
  * System stats (CPU, RAM, GPU, etc.)
* Clean and customizable

---

### 🔗 Link Detection

* Automatically captures **links shared in-world/logs**
* Saves them for later use

---

### 🔔 Smart Alerts & Sounds

* Custom sounds for:

  * Friends
  * Staff (VRChat / world staff)
  * Warning levels (red / orange / yellow)
* Detection alerts for:

  * Suspicious clients
  * Crashers

---

### 🌐 Webhook Support

* Send join/leave events to Discord
* Includes:

  * World info
  * Instance details
  * Player data

---

### 🎨 UI Customization

* Fully customizable theme:

  * Colors
  * Fonts
  * UI layout consistency across all windows 

---

### 🚀 Auto Updating System

* Checks GitHub for updates
* Downloads and installs automatically
* Keeps your tool up-to-date without manual work 

---

### 🛡️ Integrity & Safety System

* Built-in **file integrity verification**
* Detects tampering or corrupted files
* Uses signed manifest system for validation 

---

### ⚙️ Runtime Status & Error Handling

* Tracks errors, warnings, and connection states
* Prioritizes important issues automatically
* Helps keep the tool stable 

---

### 📦 Smart Startup System

* Splash screen with:

  * Auto dependency install
  * Status updates
  * Smooth UI + audio support 

---

### 📂 Open Source

* 100% transparent code
* You can verify everything yourself
* No hidden behavior

---

### 2️⃣ Extract / Unzip it


### 5️⃣ Run the tool

## 🚀 HOW TO USE

1. Start the tool
2. Launch VRChat
3. Join a world

The tool will automatically:

* Track players
* Log joins/leaves
* Update history
* Show overlay + OSC data
